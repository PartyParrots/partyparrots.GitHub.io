记一个可以实时协同的网站

· 2021-05-24 ·

和朋友开发了一个网站。每次修改html文件都要发来发去，很麻烦。正好看到 JetBrains 从V2020.02的版本就击集成了 Code With Me，就想有没有网站可以实时协同而又不用下载软件呢？

在经过一番百度搜索后，我找到了这个网站：

https://codeshare.io/

开始界面没有多余的按钮，只有三个：Sign Up Log in 和 Share Code Now

点击Share Code Now后

界面如下：

直接可以在左边的区域写代码

右边的齿轮按钮点击后，可以编辑代码样式等等。

右边的可以与好友/同事一起视频通话

点击 可以把文件保存下来

Share 则是可以把链接分享给好友，与好友一起愉快写代码。

这个网站是完完全全的节省了我和朋友之间互相发html文件（或其他的文件）所消耗的时间

唯一的缺点就是右下角会时不时弹出过广告，此外的，我都可以接受。
